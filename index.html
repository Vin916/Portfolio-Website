Hi, I'm Vineeth Reddy

Student at Rutgers University

I'm a junior at Rutgers University - New Brunswick, triple majoring in Computer Science, Data Science, and Economics. I am driven by a passion for exploring how technology can streamline processes, enhance decision-making, and solve intricate challenges across various industries. I want to apply my technical, analytical, and problem solving skills to help solve real world challenges.

In terms of career goals, I aspire to work at the intersection of software development, machine learning, and economics, leveraging data-driven insights to inform decision making and create solutions that promote sustainable growth and equitable outcomes in a rapidly evolving global economy.

I enjoy connecting with new people, especially from different backgrounds, as it allows me to exchange ideas, and gain new perspectives. Building these relationships helps me grow both personally and professionally while fostering a deeper understanding of the diverse challenges and opportunities in the world.

Courses

Data Structures

Gained expertise in analyzing algorithm efficiency and implementing complex data structures including trees, graphs, and hash tables. Developed skills in optimizing data structure performance for real-world applications and debugging implementation issues.

Linear Algebra

Mastered fundamental concepts including matrix operations, eigenvalues/eigenvectors, and vector spaces with applications to coordinate transformations. Developed proficiency in solving linear systems using Gaussian elimination and understanding geometric interpretations of linear transformations.

Discrete Structures 1/2

Built strong foundation in combinatorics, discrete probability, and graph theory essential for algorithm design and analysis. Developed expertise in counting techniques, probability distributions, and graph algorithms including path analysis and connectivity problems.

Computer Architecture

Gained comprehensive understanding of computer system design including processor architecture, memory hierarchy, and I/O systems. Developed proficiency in C programming, assembly language, digital logic design, and the critical software/hardware interface that drives modern computing systems.

Design and Analysis of Computer Algorithms

Mastered complexity analysis and algorithm design principles through studying sorting, searching, and graph algorithms including shortest path and minimum spanning tree. Developed understanding of NP-completeness, approximation algorithms, and computational complexity theory for solving real-world optimization problems.

Software Methodology

Developed expertise in large-scale Java application development using object-oriented design principles, design patterns (MVC, Singleton), and GUI programming with Swing/AWT. Gained proficiency in professional development tools including Eclipse, version control systems, unit testing, and introductory experience with mobile development and multithreading.
